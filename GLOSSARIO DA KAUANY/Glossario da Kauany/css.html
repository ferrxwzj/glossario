Tags CSS

color – Define a cor do texto do elemento. Usado para alterar a cor das letras e tornar o texto mais legível ou destacado. color: red;
background-color – Define a cor de fundo do elemento. Serve para destacar o conteúdo ou separar visualmente seções da página. background-color: #f0f0f0;
font-size – Define o tamanho da fonte do texto, controlando o quão grande ou pequeno o texto aparecerá na tela. font-size: 16px;
margin – Define a margem externa ao redor do elemento, criando espaços entre ele e outros elementos para evitar que fiquem colados. margin: 10px;
padding – Define o espaçamento interno entre o conteúdo e a borda do elemento, dando espaço para o conteúdo "respirar". padding: 5px;
border – Define a borda do elemento, incluindo sua espessura, estilo e cor, servindo para delimitar visualmente a área do elemento. border: 1px solid black;
width – Define a largura do elemento, controlando o quanto ele ocupa horizontalmente na página. width: 200px;
height – Define a altura do elemento, controlando o espaço vertical que ele ocupa. height: 100px;
display – Controla como o elemento é exibido na página, podendo ser bloco, linha ou layout flexível para organizar os filhos. display: flex;
position – Define o tipo de posicionamento do elemento, como estático, relativo, absoluto ou fixo, para controlar sua posição exata. position: relative;
top, right, bottom, left – Define o deslocamento do elemento em relação à sua posição original, usado junto com position para mover o elemento. top: 10px; left: 20px;
font-family – Define a família da fonte usada no texto, determinando o estilo das letras exibidas. font-family: Arial, sans-serif;
text-align – Alinha o texto dentro do elemento, podendo ser à esquerda, centralizado ou à direita. text-align: center;
float – Faz o elemento flutuar para a esquerda ou direita, permitindo que outros elementos se posicionem ao seu lado. float: right;
z-index – Controla a ordem de sobreposição entre elementos, definindo qual fica "por cima" quando eles se cruzam. z-index: 10;

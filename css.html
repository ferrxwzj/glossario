Tags

color – Define a cor do texto do elemento. Pode ser usada para destacar informações importantes, combinar com o design ou melhorar a legibilidade. Aceita valores em nomes (red), hexadecimais (#ff0000), RGB, RGBA, HSL e HSLA. color: red;
background-color – Define a cor de fundo do elemento. É muito usada para separar visualmente seções, criar contraste ou aplicar destaque. Aceita os mesmos formatos de cor do color. background-color: #f0f0f0;
font-size – Controla o tamanho da fonte do texto. Pode ser definido em pixels (px), em (em), porcentagem (%), rem ou até com palavras-chave como small, large etc. font-size: 16px;
margin – Define o espaço externo entre o elemento e outros ao seu redor. Pode ser aplicado de forma global (margin: 10px;) ou individualmente para cada lado (margin-top, margin-right, etc.). margin: 10px;
padding – Define o espaço interno entre o conteúdo do elemento e suas bordas. Também pode ser aplicado de forma geral ou específica para cada lado (padding-top, padding-left etc.). padding: 5px;
border – Define a borda de um elemento, controlando espessura, estilo e cor. Estilos comuns: solid, dashed, dotted, double, entre outros. border: 1px solid black;
width – Define a largura do elemento. Aceita valores em px, %, em, rem, vw (viewport width) e outras unidades. width: 200px;
height – Define a altura do elemento. Aceita os mesmos formatos de unidade do width. height: 100px;
display – Controla como o elemento é exibido na página, podendo ser bloco, linha ou layout flexível para organizar os filhos.Controla como o elemento é renderizado. Valores comuns: block – quebra a linha e ocupa toda a largura disponível. inline – fica na mesma linha e só ocupa o espaço do conteúdo. flex – cria um container flexível para organizar elementos. grid – organiza usando o sistema de grades. display: flex;
position – Define o método de posicionamento do elemento: static (padrão) – segue o fluxo normal. relative – posicionado em relação à sua posição original. absolute – posicionado em relação ao elemento pai com position diferente de static. fixed – fixo na tela mesmo ao rolar a página. sticky – alterna entre relative e fixed conforme o scroll. position: relative;
top, right, bottom, left – Ajustam o deslocamento do elemento em relação ao seu ponto de referência (definido pelo position). top: 10px; left: 20px;
font-family – Define a família tipográfica usada no texto. É recomendado colocar fontes alternativas como fallback, separadas por vírgula. font-family: Arial, sans-serif;
text-align – Alinha o texto horizontalmente dentro do elemento. Valores comuns: left, center, right, justify. text-align: center;
float – Faz o elemento “flutuar” para a esquerda (left) ou direita (right), permitindo que outros elementos ocupem o espaço ao lado. Hoje em dia, é menos usado, pois flex e grid são mais modernos. float: right;
z-index – Define a ordem de sobreposição dos elementos. Quanto maior o valor, mais “por cima” o elemento ficará quando houver sobreposição. Funciona apenas com elementos posicionados (relative, absolute, fixed ou sticky). z-index: 10;
